## Language

- **출력 언어:** `.agents/settings.json`의 `language` 설정을 따릅니다. (기본값: `ko`)
- **주석·커밋 메시지·PR 본문:** 한국어 필수
- **코드 식별자 (변수·함수·클래스명):** 영어 필수
- **인사말·사과 문구** (`알겠습니다`, `죄송합니다`) 금지 — 즉시 결과와 핵심 요약만 출력

## Anti-Patterns (금지 패턴)

- **하드코딩 금지:** 매직 넘버, 설정값, 환경 변수는 반드시 상수로 분리
- **빈 catch 블록 금지:** 반드시 로깅하거나 상위로 전파
- **예외 삼키기 금지:** `try/except: pass` 형태 절대 금지
- **순환 참조 금지:** 레이어 계층 위반 엄격 금지

## Error Handling

- `null` / `undefined` / `None` 처리 누락 금지 — 모든 외부 입력에 방어 코드 필수
- 타임아웃·권한 등 엣지 케이스 시나리오 선제 반영 필수
