# 실행 기록

이 파일은 `python run-evals.py run --project-root <PATH>` 실행 시 자동으로 재생성됩니다. **직접 편집하지 마세요** — 다음 실행에서 덮어써집니다.

- 원본 데이터: `logs/history/*.json` (스킬 실행마다 1개씩 누적되는 원본 로그)
- `logs/latest.json`: 가장 최근 실행 1건의 스크래치 파일 (채점 후 위 history/로 복사되며, 다음 실행 시 덮어써짐)
- 이 `history.md`: 위 `history/*.json` 전체를 모아 사람이 한 파일로 훑어볼 수 있게 만든 표
